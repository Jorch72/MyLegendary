I saw a fanmade program in GameDev Studio that semi worked and since I'm a programmer, I thougth "I can do better!", so this is my attempt at making a working game of Legendary.
I love and own everything Legendary!
So the idea is to make a program that can play each game individually, but can also randomize (or give you the option to choose) with what cards and rules to play from each game.
Games that will be implemented:
-All Marvel
-Buffy
-Big Trouble in Little China
-Aliens +Expansions
-Predator
-Matrix (Probably just the cards)
-Firefly (Probably just the cards)
-James Bond +Expansions
-X-Files (Probably just the cards)
-Game of Thrones (Cards will be implemented, but not the competitive system it currently has)
